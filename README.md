# Agrosanoatni rivojlantirish agentligi

Страница с ссылками на социальные сети для Agrosanoatni rivojlantirish agentligi | QXV huzurida

## Технологии

- Next.js 14
- TypeScript
- TailwindCSS
- React 18

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка для продакшена

```bash
npm run build
npm start
```
